# Customer Lead Research

This file lists public lead targets that the judge/mentor companies could sell
to. It is intentionally customer-side: these are not internal stakeholders at the
represented companies.

Important caveats:

- Do not claim any target is or is not already using the represented company's
  product unless a public source proves it.
- No private contact data is included.
- Person entries are public KOL/influencer targets only, mainly for Nabla.
- The code-ready version is in [`data/customer-leads.ts`](../data/customer-leads.ts).

## Best Stage Packs

Use these if the demo needs only five strong examples:

1. **Nabla -> Mayo Clinic / AP-HP / Eric Topol**
   Healthcare trust, clinician workflow, and public proof are easy to explain.
2. **Photoroom -> Shopify / Etsy / Faire**
   Visual ecommerce workflow is obvious on stage.
3. **Airtable -> Decathlon / BlaBlaCar / Back Market**
   Enterprise workflow and marketplace operations are concrete.
4. **FullEnrich + Sillage -> Lovable / Mistral AI / Hager Group**
   Shows sponsor APIs in the core loop: signals then enrichment.
5. **Dataiku or Accenture -> Sanofi / BNP Paribas / LVMH**
   Large-enterprise AI transformation and account complexity.
